Remez exchange algorithm

The "Remez exchange algorithm" finds a polynomial approximation to a given function over an interval by minimizing the maximal absolute difference. See https://en.wikipedia.org/wiki/Remez_algorithm for background information.
Mandatory positional Inputs:
  • fun: function to be approximated
  • fun_der: derivative of fun
  • interval: Start and end point of the interval over which to approximate as 2-element vector.
  • n: Order n of the polynomial used for approximation
Optional inputs given in name-value pairs:
  • maxIter: maximum number of iterations
  • thresh: threshold of point shift to stop. If single point update is below threshold, iterations are stopped.
  • multi: flag controlling if all (true) or one (false) node is exchanged in each iteration. default: false. The 'multi point exchange' converges faster but may not converge as reliable.
  • verbosity:
  • 0: no output
  • 1: little output per iteration and printing of final results
  • 2: all intermediate calculations
  • 3: all intermediate calculations including plot figures
Return values:
  • p: Coefficients for the polynomial ordered by decreasing powers of x (usual matlab convention). Approximated function value at x may be evaluated with matlab function polyval(p, x)
  • E: the maximum absolute difference (approximation error)

Helper funtions

Print an array to screen in compact form

function disp_arr(arr)
s = arrayfun(@(x) sprintf("%g", x), arr);
disp(join(s, ", "));
end

Plot the approximation and the shift of the node for the next iteration

function plot_approx(fun, interval, i, x_old_nodes_all, x_nodes_old, x_nodes_new, z)
figure(i);
x_val = linspace(interval(1), interval(2), 100);
f_val = fun(x_val);
plot(x_val, f_val);
hold on;
stem(x_old_nodes_all, fun(x_old_nodes_all), 'bo'); % old nodes
stem(x_nodes_old, fun(x_nodes_old), 'go'); % old node(s) to be moved
stem(x_nodes_new, fun(x_nodes_new), 'ro'); % new node(s) after move
plot(z, z*0, 'b|', 'MarkerSize', 20); % markers at interval limits
hold off;
title("iteration " + i);
xlim([interval(1) - 0.1*(interval(2)-interval(1)), ...
interval(2) + 0.1*(interval(2)-interval(1))]);
grid on;
end
